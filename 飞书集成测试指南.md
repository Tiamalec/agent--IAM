# 飞书集成测试指南

本文档提供飞书AI Agent集成功能的详细测试指南，包括测试环境配置、测试用例说明、测试步骤及预期结果。

## 1. 测试环境配置

### 1.1 硬件要求

| 组件 | 最低要求 | 推荐配置 |
|------|----------|----------|
| 处理器 | 4核 | 8核 |
| 内存 | 8GB | 16GB |
| 磁盘空间 | 10GB | 20GB |
| 网络 | 10Mbps | 100Mbps |

### 1.2 软件要求

- **Python版本**：Python 3.8+
- **操作系统**：macOS、Linux、Windows
- **依赖包**：详见 requirements.txt

### 1.3 环境变量配置

```bash
# 飞书应用配置（用于真实API测试）
export FEISHU_APP_ID=your_app_id
export FEISHU_APP_SECRET=your_app_secret
export FEISHU_SKILLS_ENABLED=true

# 同步配置
export FEISHU_SYNC_DEPARTMENTS=true
export FEISHU_SYNC_USERS=true
export FEISHU_INCREMENTAL_SYNC=true
export FEISHU_SYNC_BATCH_SIZE=100
export FEISHU_SYNC_INTERVAL=3600

# 日志配置
export LOG_LEVEL=INFO
export LOG_DIR=./logs

# 测试模式（使用模拟API）
export FEISHU_SKILLS_ENABLED=false
```

### 1.4 安装测试依赖

```bash
# 安装项目依赖
pip install -r requirements.txt

# 安装测试依赖
pip install pytest pytest-cov pytest-asyncio

# 安装Streamlit（用于可视化测试）
pip install streamlit plotly pandas
```

python -m pytest tests/test_feishu_integration.py -v
