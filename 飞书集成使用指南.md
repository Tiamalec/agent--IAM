# 飞书集成使用指南

本文档提供飞书AI Agent集成功能的详细使用指南，包括安装部署、配置说明、基本操作流程、功能示例及故障排除。

## 1. 功能特性概述

### 1.1 核心功能

飞书AI Agent集成功能为AI Agent IAM系统提供了与飞书平台的无缝集成，主要功能包括：

1. **Skill-based权限映射**：19个飞书Skills与IAM权限的精细映射
2. **Token委托链**：管理员Token → Agent Token → Skill Token的安全委托
3. **统一审计**：所有飞书操作记录到IAM审计日志
4. **细粒度控制**：基于Skill、Action、Resource的三层权限控制
5. **组织架构同步**：自动同步飞书部门和用户到IAM系统

### 1.2 支持的飞书Skills

| Skill | 功能 | 支持的操作 |

## 3. 基本操作流程

### 3.1 快速开始

#### 3.1.1 使用模拟API（无需配置飞书应用）

```python
from agent_iam.feishu_integration import FeishuIntegration, FeishuSkill
from agent_iam.models import Actor, ActorType

# 初始化飞书集成（使用模拟API）
integration = FeishuIntegration()

# 创建飞书AI Agent
calendar_agent = Actor(
    name="智能日程助手",
    type=ActorType.MASTER_AGENT
)

# 注册飞书Agent并获取Skill Tokens
skill_tokens = integration.register_feishu_agent(
    calendar_agent,
    [FeishuSkill.CALENDAR, FeishuSkill.IM]
)

print(f"注册的Skill Tokens: {skill_tokens}")
```

#### 3.1.2 使用真实飞书API

```python
import os

# 设置环境变量
os.environ["FEISHU_APP_ID"] = "your_app_id"
os.environ["FEISHU_APP_SECRET"] = "your_app_secret"
os.environ["FEISHU_SKILLS_ENABLED"] = "true"

# 初始化飞书集成（使用真实API）
integration = FeishuIntegration()

# 执行飞书命令
result = integration.execute_feishu_command(
    skill_token="your_skill_token",
    command="calendar agenda",
    params={"date": "2024-01-15"}
)

print(f"命令执行结果: {result}")
```

## 4. 高级功能

### 4.1 权限管理

#### 4.1.1 权限映射

```python
from agent_iam.feishu_integration import FeishuPermissionMapper, FeishuSkill, FeishuAction, FeishuResource

# 创建权限映射器
mapper = FeishuPermissionMapper()

# 将飞书权限映射到IAM Scopes
iam_scopes = mapper.map_feishu_to_iam(
    FeishuSkill.CALENDAR,
    FeishuAction.READ,
    FeishuResource.CALENDAR
)
print(f"IAM Scopes: {iam_scopes}")

# 验证IAM Scopes是否足够
is_valid = mapper.validate_iam_scopes_for_feishu(
    FeishuSkill.CALENDAR,
    FeishuAction.READ,
    {"read:calendar", "read:event"}
)
print(f"权限验证结果: {is_valid}")
```

#### 4.1.2 自定义权限映射

```python
# 添加自定义映射
mapper.add_custom_mapping(
    FeishuSkill.CALENDAR,
    FeishuAction("custom_action"),
    {"custom:scope1", "custom:scope2"}
)

# 从配置文件加载
mapper.load_config("/path/to/config.json")
```

### 4.5 错误处理

```python
from agent_iam.feishu_integration import FeishuAPIClient, handle_feishu_error, FeishuError, FeishuAuthError

client = FeishuAPIClient()

try:
    result = client.execute_command("calendar", "agenda", {"date": "2024-01-15"})

    # 检查是否有错误
    error = handle_feishu_error(result)
    if error:
        if isinstance(error, FeishuAuthError):
            print(f"认证错误: {error.message}")
        else:
            print(f"API错误: {error.message}")
        return

    # 处理成功结果
    print(f"执行成功: {result}")

except FeishuError as e:
    print(f"飞书错误: {e.code} - {e.message}")
except Exception as e:
    print(f"系统错误: {e}")
```

## 7. 常见问题处理

### 7.1 认证问题

#### 问题：飞书API认证失败

**症状**：
```
FeishuAuthError: 认证失败: Invalid app_id or app_secret
```

**解决方案**：
1. 检查环境变量配置
```bash
export FEISHU_APP_ID=your_correct_app_id
export FEISHU_APP_SECRET=your_correct_app_secret
```

2. 确认飞书应用已启用相关权限
3. 检查应用是否已发布

### 7.2 同步问题

#### 问题：组织架构同步失败

**症状**：
```
SyncError: 获取部门列表失败
```

**解决方案**：
1. 检查网络连接
2. 确认飞书应用具有通讯录读取权限
3. 增加同步重试次数
```python
config = SyncConfig(
    max_retries=5,
    retry_delay=10
)
```

### 7.3 缓存问题

#### 问题：缓存数据过期但仍被使用

**解决方案**：
1. 手动清空缓存
```python
client.clear_cache()
```

2. 调整缓存TTL
```python
client.cache_ttl = 60  # 设置60秒TTL
```

### 7.4 性能问题

#### 问题：API响应缓慢

**解决方案**：
1. 启用缓存减少重复请求
2. 使用增量同步替代全量同步
3. 增加批处理大小
```python
config = SyncConfig(batch_size=200)
```

### 7.5 权限问题

#### 问题：权限验证失败

**症状**：
```
PermissionDenied: 用户缺少必要权限
```

**解决方案**：
1. 检查用户IAM Scopes配置
2. 确认Skill已正确注册
3. 使用权限映射工具验证映射规则
